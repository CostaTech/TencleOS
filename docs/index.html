Personalizzazione Sintassi

Modifica config.py per cambiare i comandi:

# Esempio: cambiare il comando print
COMANDO_PRINT = "stampa"  # Invece di "int << func >>"

# Ora puoi usare:
stampa "Ciao!"

Comandi personalizzabili:

  • COMANDO_PRINT - Stampa output
  • COMANDO_IF - Condizione if
  • COMANDO_ELIF - Condizione elif
  • COMANDO_ELSE - Condizione else
  • COMANDO_WHILE - Loop while
  • COMANDO_FOR - Loop for
  • COMANDO_IN - Operatore in
  • COMANDO_IMPORT - Import librerie
  • COMANDO_TRUE - Booleano true
  • COMANDO_FALSE - Booleano false
  • COMANDO_AND - Operatore AND
  • COMANDO_OR - Operatore OR
  • COMANDO_NOT - Operatore NOT
  • COMANDO_BREAK - Break loop
  • COMANDO_CONTINUE - Continue loop

Gioco Snake (Semplificato)

use pygame

pygame.init()
screen = pygame.display.set_mode([600, 400])
pygame.display.set_caption("Snake Game - TLang")
clock = pygame.time.Clock()

x = 300
y = 200
running = on

<<While>>! <on> running {
    events = pygame.event.get()
    
    for e in events {
        << ! >func> e.type == pygame.QUIT {
            running = off
        }
    }
    
    screen.fill([0, 0, 0])
    pygame.draw.rect(screen, [0, 255, 0], [x, y, 20, 20])
    pygame.display.flip()
    clock.tick(60)
}

pygame.quit()
